This file explains the various data files and the sources.

Generated Data:
nfl_player_stats.csv - This is generated when you run the nfl_player_download.py
nfl_team_data.csv - This is generated when you run the nfl_team_download.py
all_games_weather.csv - This is a combination of data downloaded and generated from the nfl_weather.py
game_id_map.csv - File used to map the datasets with the different game ids
model_results.csv - File generated that records the results of running the Jupyter Notebook

Downloaded Data:
NFL Play Statistics dataset (primary)
URL: https://www.kaggle.com/toddsteussie/nfl-play-statistics-dataset-2004-to-present
Files: games.csv, officials.csv

Detailed NFL Play-by-Play Data 2009-2018
URL: https://www.kaggle.com/maxhorowitz/nflplaybyplay2009to2016?select=NFL+Play+by+Play+2009-2018+%28v5%29.csv
File: NFL Play by Play 2009-2018 (v5).csv renamed to nfl_play_by_play09-18.csv

NFL scores and betting data
URL: https://www.kaggle.com/tobycrabtree/nfl-scores-and-betting-data
File: nfl_teams.csv, spreadspoke_scores.csv

NFL-Weather-Project
URL: https://github.com/Nolanole/NFL-Weather-Project
File: all_games_weather.csv

nflscrapR-Data
URL: https://github.com/ryurko/nflscrapR-data
File: game_data folder to get the game IDs
